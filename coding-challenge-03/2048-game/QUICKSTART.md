# 2048 Game - Quick Start Guide

## Play Right Now!

```bash
cd coding-challenge-03/2048-game
open index.html
```

That's it! The game will open in your default browser and you can start playing immediately.

## Controls

- **Arrow Keys** (‚Üë ‚Üì ‚Üê ‚Üí): Move tiles
- **WASD Keys**: Alternative movement (W=up, S=down, A=left, D=right)  
- **R Key**: Quick restart
- **New Game Button**: Restart the game

## Objective

Slide tiles to merge matching numbers. Get to 2048 to win!
